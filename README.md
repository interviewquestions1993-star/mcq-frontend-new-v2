## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/mcq-exam-preparer.git
   cd mcq-exam-preparer
   ```

2. **Install frontend dependencies**
   ```bash
   cd Frontend
   npm install
   ```

3. **Install backend dependencies**
   ```bash
   cd ../Backend
   pip install -r requirements.txt
   ```

4. **Start the development servers**

   Frontend:
   ```bash
   cd Frontend
   npm start
   ```

   Backend:
   ```bash
   cd Backend
   python main.py
   ```

5. **Open your browser**
   Navigate to `http://localhost:4200`

## 🚀 Deployment

### Frontend Deployment (Vercel/GitHub Pages)
```bash
npm run build:seo

```

The backend is configured for Render deployment with the following files:
